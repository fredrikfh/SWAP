# Swap

###### Check the latest build: https://swap-93d9f.web.app/

### To run the app
* run `cd swap`
* run `yarn`
* run `yarn start` (or npm start)


### To run checks:

* `npx prettier --check .`
* `npx prettier --write .`
* `npx eslint --fix .`


### Adding a new package
* run `yarn add <package>`
* run `yarn` (updates yarn.lock)
* Push `yarn.lock`
